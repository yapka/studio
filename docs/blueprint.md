# **App Name**: GeoReact White

## Core Features:

- Map Display: Display an interactive map using a library like Leaflet or Mapbox GL.
- Layer Management: Allow users to add, remove, and reorder map layers.
- Data Upload: Enable users to upload geospatial data in formats like GeoJSON, KML, or Shapefile.
- Data Visualization: Provide options to style and visualize the uploaded data with different colors, symbols, and labels.
- Search and Geocoding: Integrate a search function that uses a geocoding service to find locations by address or name.
- Location Awareness: Show current location on the map.

## Style Guidelines:

- Primary color: Light gray (#F0F0F0) for a clean and modern look.
- Background color: White (#FFFFFF) for a simple interface.
- Accent color: Soft blue (#A0C4FF) for interactive elements and highlights.
- Body and headline font: 'Inter', sans-serif, for a clean and readable text.
- Use simple, clear icons to represent map controls and features.
- Maintain a clean and intuitive layout with a focus on map display area.
- Subtle animations for map interactions and transitions.
